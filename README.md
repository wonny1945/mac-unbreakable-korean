## Project Overview

This project is a web-based tool designed to solve encoding issues for Korean language files created on Mac operating systems. It aims to resolve encoding mismatches that occur when sharing text files between Windows and Mac, facilitating smooth file exchange.

## Key Features

- Encoding conversion for Korean text files created on Mac
- Client-side processing to maintain file security
- Intuitive and user-friendly interface

## Tech Stack

- **TypeScript**: For type safety and improved development productivity
- **SvelteKit**: For fast and efficient web application development
- **Shadcn-svelte**: For modern and customizable UI components

## Deployment

This project is deployed using Vercel.

Deployment URL: [https://mac-unbreakable-korean.vercel.app/](https://mac-unbreakable-korean.vercel.app/)

## Security and Privacy

This application uses complete client-side processing. Users' files or data are not transmitted to any server, and all processing occurs within the user's browser. This ensures that file security and privacy are safely protected.
